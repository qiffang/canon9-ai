# Three-Timing Consolidation

engram9 consolidates memory at three moments, each at a different granularity:

1. **Encoding** (ingest agent): New information is immediately woven into existing wiki pages. The ingest agent reads the index, identifies the 1-3 most relevant pages, and integrates new facts with source provenance.

2. **Retrieval** (query agent): Every recall is reconstructive. The query agent synthesizes across pages, fixes errors it encounters, and may archive new insights discovered during retrieval.

3. **Sleep** (compile agent): Periodic global consolidation distills episodic experiences into semantic facts, detects contradictions, prunes stale pages, and strengthens cross-links.

This mirrors neuroscience research on hippocampal replay and neocortical consolidation (Tononi & Cirelli, 2003).
